# Reset CrossOver Bottle — How to Use

## First-time setup

1. Open **Terminal** (Applications → Utilities → Terminal).
2. Type `chmod +x ` (with a trailing space — don't hit enter yet).
3. Drag `Reset_CrossoverBottle.command` from Finder into the Terminal window.
4. Hit **Enter**.

Done. Now just double-click `Reset_CrossoverBottle.command` any time you want to reset the Steam bottle.
